# REAP Landing Page

A minimalist, dark-themed landing page for a real estate asset protocol (REAP) featuring interactive 3D backgrounds and a modern obsidian design system.

## Getting Started

This project is optimized for **Bun** for fast dependency installation and performance.

### Prerequisites

- [Bun](https://bun.sh/) (recommended) or Node.js

### Run Locally

1. **Install dependencies:**
   ```bash
   bun install
   ```

2. **Run the development server:**
   ```bash
   bun dev
   ```

3. **Open the app:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

- `bun dev`: Runs Next.js with Turbopack for faster development.
- `bun build`: Builds the application for production.
- `bun start`: Starts the production server.
- `bun lint`: Runs ESLint for code quality checks.
- `bun clean`: Cleans the build output.
